# EEPROM BR93L56FW GPIO-based SPI Driver

A software-emulated (bit-banging) SPI driver for the BR93L56FW EEPROM implemented using GPIO.

### Features
* **Software SPI (Bit-Banging):** Implements SPI signaling using GPIO for systems without a hardware SPI controller.
* **Core Functions:** Supports basic EEPROM operations, including byte/word Read and Write.
* **Status:** This project is in a **Feature-Complete** state. The core functionality has been fully tested and validated, and no further active development is planned.
